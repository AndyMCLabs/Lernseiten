Was ist JavaScript?

JavaScript ist die Magie hinter der Bewegung im Web. Während HTML das Skelett deiner Seite bildet und CSS ihr schicke Rüstung verpasst, ist JavaScript der Zauberstab, der alles lebendig macht - von klickbaren Buttons bis zu epischen Animationen und sprechenden Toast-Messages.

Entwickelt wurde JavaScript 1995 von Brendan Eich - in gerade einmal 10 Tagen (True Story!). Heute ist es eine der wichtigsten Programmiersprachen der Welt, wenn’s ums Web geht. Du willst Formulare prüfen, auf Klicks reagieren oder ein Mini-Spiel im Browser zaubern? JavaScript ist dein treuer Begleiter.

Und nein, JavaScript hat nix mit Java zu tun - so wie ein Donut auch nix mit einem Donau-Dampfschiff zu tun hat. 😅

🧪 Einfaches Beispiel

Hier siehst du ein kleines Beispiel: Ein Button, der beim Klicken eine Nachricht zaubert!

<button onclick="sagHallo()">Klick mich!</button>

<script>
function sagHallo() {
alert("Hallo, du mutiger JavaScript-Held!");
}
</script>

⚔️ Übungen (vom Lehrling zum Meister)

  2. Schreibe eine Funktion, die eine Nachricht in der Konsole anzeigt.

  4. Erstelle ein Formular mit einem "Absenden"-Button und überprüfe mit JavaScript, ob das Eingabefeld leer ist.
