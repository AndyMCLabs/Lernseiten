⚔️ HTML, CSS & JavaScript - Das magische Dreieck der Webentwicklung

Stell dir vor, du baust ein Haus in Minecraft 🏠 - du brauchst erstmal die Grundstruktur: Wände, Fenster, Türen - das ist HTML.

Danach willst du es hübsch machen: bunte Tapeten, coole Möbel, vielleicht ein paar Pflanzen - das ist CSS.

Aber das Haus soll auch leben: Türen gehen auf, Lichter flackern, NPCs laufen herum - das ist JavaScript! 🧠✨

Diese drei arbeiten als Dream Team zusammen - einzeln können sie was, aber erst gemeinsam entfalten sie ihre volle Power:

  • HTML = Struktur (Was ist auf der Seite?)
  • CSS = Design (Wie sieht es aus?)
  • JavaScript = Interaktion (Was kann ich damit tun?)

💡 Praktisches Beispiel: Interaktiver Button mit Stil

Hier zeigen alle drei, was sie draufhaben:

<!DOCTYPE html>
<html lang="de">
<head>
  <meta charset="UTF-8">
  <title>Dreifaltigkeit des Webs</title>
  <style>
    button {
      background-color: #4CAF50;
      color: white;
      padding: 10px;
      border: none;
      border-radius: 5px;
      font-size: 16px;
    }
  </style>
</head>
<body>

  <button onclick="alert('JS hat gesprochen!')">Klick mich!</button>

</body>
</html>

HTML = Button-Struktur 💪, CSS = Style 💅, JavaScript = Funktion 💥.

⚔️ Übungen - Der Pfad zur Meisterschaft

  • Erstelle eine HTML-Seite mit einer Überschrift und einem Button.
  • Style den Button mit CSS (z.B. Farbe, Rundung, Hover-Effekt).
  • Füge per JavaScript eine Nachricht hinzu, wenn der Button geklickt wird.
  • Füge eine CSS-Animation ein, wenn der Button angeklickt wird.
  • Erstelle eine "Like"-Funktion: Jeder Klick zählt Likes hoch und zeigt die Anzahl an.
