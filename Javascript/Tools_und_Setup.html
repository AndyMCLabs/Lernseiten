Tools & Setup für JavaScript

Bevor du loslegst, brauchst du keine magischen Artefakte - nur ein paar gute Werkzeuge, die dich beim Coden unterstützen. Und das Beste: Die meisten davon sind kostenlos!

🔧 Die Ausrüstung eines JavaScript-Helden

  • Code-Editor: Visual Studio Code (VS Code) ist der König unter den Editoren. Kostenlos, leichtgewichtig und mit tausenden Erweiterungen.
  • Browser: Chrome, Firefox oder Edge - mit integrierten Entwickler-Tools. Sie sind wie ein Zauberspiegel, der dir zeigt, was dein Code gerade treibt.
  • Terminal (optional): Für mutige Helden, die sich auch mit der Kommandozeile anfreunden wollen - nützlich für npm, Git & Co.
  • Dateiverwaltung: Ein sauberer Projektordner mit einer HTML-, CSS- und JS-Datei. Ordnung ist das halbe Leben (und spart viele Bugs).
  • Live Server: Eine Erweiterung für VS Code, die deine Seite bei jeder Änderung automatisch neu lädt - wie ein Zauber-Refresh!

⚙️ Minimales JavaScript-Setup

Du brauchst nur drei Dateien:

projektordner/
├── index.html
├── style.css
└── script.js

Und hier die Verbindung der drei:

<!DOCTYPE html>
<html lang="de">
    <head>
        <meta charset="UTF-8">
        <title>Mein erstes JS-Projekt</title>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <h1>Hallo JavaScript!</h1>
        <script src="script.js"></script>
    </body>
</html>

🗡️ Übungen (Leveln durch Setup-Mastery)

  1. Installiere Visual Studio Code und richte ein neues Projekt ein.
  2. Erstelle die Dateien index.html, style.css und script.js.
  3. Schreibe ein console.log("Hello World") in die JS-Datei und öffne die Konsole im Browser.
  4. Installiere die Live Server Extension in VS Code und nutze sie.
  5. Füge einen Button in HTML ein, der beim Klicken eine JS-Funktion ausführt.
